# 桌面宠物Alex - DesktopPetAlex
概述：Alex的应援产物可以直接于 下载并双击exe文件运行（无需安装java）
# 功能描述：
* Alex的眼睛可以随鼠标移动
* 点击月亮可以直接进入Alex直播间
* 鼠标点击Alex，Alex的耳朵会立起来噢~并且可以与Alex进行交互
* <del>Alex的尾巴可以摇晃</del>(后因不够美观，删除)
* 可以通过拖拽，把Alex放置在任何一个你喜欢的地方
